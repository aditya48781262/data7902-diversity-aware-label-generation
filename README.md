# data7902-diversity-aware-label-generation
Project researching the ability of two LLMs Mistral 7B Instruct and Llama 3.3 70B Instruct to generate high quality, diverse and unbiased labels for a misinformation dataset. This project was undertaken over the course of two semesters, DATA7901 and DATA7902. Labelled datasets are provided in the datasets folder and python scripts are provided in the scripts folder.

Note that to run this code, an AWS Bedrock subscription is necessary.
